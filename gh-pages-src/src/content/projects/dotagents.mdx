# dotagents: Claude Code AI Agent Management System

Elevating human-AI collaboration through standardized agent personas and project-specific knowledge-base management.

## Strategic Impact

dotagents addresses the common friction in human-AI collaboration: consistency and context. By providing a management layer for AI agents, it allows engineering teams to:

- **Standardize Agent Personas**: Ensure every AI agent follows identical architectural standards (SOLID, VSA) regardless of the developer's local environment.
- **Distribute Knowledge**: Share project-specific context and "memory" across the team to reduce onboarding time for both humans and agents.
- **Accelerate Development**: Reduce the cognitive load of repeated specification by maintaining a persistent library of high-rigor agent definitions.

## Technical Excellence

The dotagents ecosystem is built using **Astro v5** and **TypeScript**, ensuring a lightweight, high-performance footprint. It implements strict **Vertical Slice Architecture** to maintain high cohesion between agent definitions and their supporting tools.

Every component of dotagents follows the same **TDD (Atomic Verification)** principles that define the `iamrichardD` engineering standard, ensuring that the infrastructure layer is as reliable as the code it helps generate.
